# AMFI NAV Data Extractor

Downloads mutual fund NAV data from AMFI India and saves it to `amfi_nav_data.tsv`.

## How to Run

**Windows:**
```powershell
bash scheme.sh
```

**Linux/Mac:**
```bash
chmod +x scheme.sh
./scheme.sh
```

## Output

Creates `amfi_nav_data.tsv` with Scheme Names and Net Asset Values for 14,000+ mutual funds.
